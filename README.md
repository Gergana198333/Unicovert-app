# Unicovert-app

A simple, responsive web application for converting units across different categories.

## Features

- **Multiple Categories**: Length, Weight, Temperature, and Volume conversions
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Swap Functionality**: Quickly reverse the conversion units
- **Clean UI**: Modern, intuitive interface with smooth interactions
- **Conversion History**: Keeps track of recent conversions (coming soon)

## Project Structure


## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS variables and Grid/Flexbox
- **JavaScript (ES6+)**: Application logic (to be implemented)

## Getting Started

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Select a category from the dropdown
4. Enter a value and select the units to convert
5. The result will be displayed automatically
